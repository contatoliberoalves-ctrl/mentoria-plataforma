# Mentoria OAB — Plataforma de Estudos

Plataforma de estudos para a 1ª fase do Exame da OAB com cronograma adaptativo,
banco de questões, caderno de erros, gamificação e painel administrativo multiusuário.

## Como usar

Acesse o link público gerado pelo Cloudflare Pages / Netlify (veja abaixo).

## Stack

- React via CDN (sem build)
- Supabase (Auth + Postgres) — plano gratuito
- Hospedagem estática gratuita
